Unit Test The process where you test the smallest functional unit of code
Smoke Test It's a basic kind of test that needs to work before people go on further testing.
Functional Testing Functional testing is a type of test that looks for whether each application feature work correctly.
Test Driven Development TDD is a method of coding in which you first write a test, and it fails then you write the code to pass the test of development and clean up the code.
Non-Functional Testing Non-functional Testing is a type of Software Testing that is performed to verify the non-functional requirements of the application. Non-functional testing is a type of software testing that focuses on rate
Load Test Load testing is a simulated load on your web that determines the behavior of the application when multiple users use it at the same time.
Regression Test Ensures that new code change haven't introduced new bugs in existing functionality.
Acceptance Testing Acceptance testing is a quality assurance process that verifies a system or application meets its requirements and is ready for use.
code framework A code framework is a pre-written set of code that helps developers build applications.
Front-end JavaScript frameworks Sets of pre-written code that provide developers with a sealable and maintainable structure for creating web user interfaces more efficiently.
Content Management Systems CMS is a system that helps administrators add, manage, edit and delete content on websites.
