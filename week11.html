Encapsulation-OOP It hides the code but still allows you to use the parts that we make visible.
Abstraction-OOP Abstraction is a way programmers hide complexity in programming objects.
instance Instance is best understood as it relates to "class" in programming.
JSON (JavaScript Object Notation) JSON is a plain text written language that is used for sending data between computers. JSON it can also give value to variables
constructor Constructor is a special function inside a class that runs automatically when you create a new object.
Python self Self represents the instance of the class by using the self we can access attributes and methods of the class in python.
JavaScript this this keyword refers to the context where a piece of code such as a functions body is supposed to run.
imperative programming A software development paradigm where Functions are implicitly coded in every step required to solve a problem.
declarative programming A high-level programming concept, which is the opposite of imperative programming.
Array.map() Array.map() is a method that creates a new array by applying a function to each element of the original array.
Polymorphism-OOP Javascript polymorphism allows different classes to define the same method but with different executions.
model-view-controller design pattern MVC is a software design patter commonly used for developing user interfaces that divides the related program logic.
table,tr,th,td tags An HTML tag consists of one table element and one or more tr,th and td element.
