Week 13 vocab

Uniform Resource Locator (URL) https:// is the protocol, www is the subdomain, site name EX : google is the domain and .com/.io ect is the top level domain.
Firewall A firewall is something that helps the computer from viruses and other stuff.
Transmission Control Protocol-Internet Protocol (TCP-IP) TCIP is where the data runs and delivers data.
Network Protocols A network protocol is a set of rules that govern how data is transmitted and formatted across a network, enabling communication between devices.
Computer Network A computer network is a system that connects two or more computing devices to share data, resources, and information, utilizing wired or wireless connections and communication protocols.
IPv4 IPv4 is like the address system for the internet.Every device that connects to the internet, like your phone or laptop gets a unique IPv4 address kind of like a home address so data knows where to go.
IPv6 Internet Protocol version 6 is the most recent version of the Internet Protocol, the communications protocol that provides an identification and location system for computers on networks and routes traffic across the Internet.
Domain Name Services (DNS) Turns domain names into IP addresses, which allow browsers and other internet resources.
Hyper-text transfer protocol (HTTP/HTTPS) A protocol that defines how web browsers and servers exchange data.
File Transfer Protocol File Transfer Protocol (FTP) is a standard network protocol used to transfer files between a client and a server over a TCP-based network, such as the internet.
Cloud Computing the on-demand delivery of computing resources, like storage, processing power, and software, over the internet, allowing users to access and use them without needing to purchase and maintain physical infrastructure.
OSI Model The OSI Model, created by the ISO (International Organization for Standardization), standardizes protocols to enable communication between different computer systems.

Virtual Private Network (VPN) A Virtual Private Network (VPN) creates a secure, encrypted connection between your device and the internet, masking your IP address and encrypting your data to enhance privacy and security, especially when using public Wi-Fi.
