Meta Vocab

Article Tag

Specifies independent, self-contained content.

<article> <h2>Google Chrome</h2> <p>Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!</p> </article>

Section Tag

Defines a section in a document.

<section> <h2> Topic </h2> <p> </p> </section>

Audio Embed Tag

Used to embed sound content in a document, such as music or other streams.

<audio controls> <source src="horse.ogg" type="audio/ogg"> <source src="horse.mp3" type="audio/mpeg"> </audio>

Video and audio source Tag

Specifies one or more media resources for the picture, radio and video tags.

<video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> < source src="movie.ogg" type="video/ogg"> </video>>

Meta Tag For Viewport

Viewport gives the browser instructions on how to control the webpages dimensions and scaling.

<meta name="viewport" content="width=device-width, initial-scale=1" />

Figure Tag

The figure tag is used to show important stuff like information and images.

< figure> <img src="pic_trulli.jpg" alt="Trulli" style="width:100%"> </figure>

Video Tag

Embeds a media player which supports video playback into the document.

<video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> < source src="movie.ogg" type="video/ogg"> </video>>

Meta Tag for Keyword

HTML tags that allow you to specify keywords for your website.

<meta name="keywords" content="HTML, CSS, JavaScript">

Aside

The aside element represents a section of a page that consists of content.

<aside> <h4>Topic</h4> <p>....</p> /aside

Figcaption tag

defines a caption for a figure element.

<figure> <img src=... alt=... style=...> <figcaption...</figcaption> </figure>
